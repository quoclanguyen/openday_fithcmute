# Open Day - FITHCMUTE
Showcase games for Open Day Event using basic AI search algorithms.
# Requirements
### Python version
Python version 3.7 or above
### Packages/Modules
`pip install tkinter`  
`pip install pygame`  
`pip install pillow`  
`pip install opencv-python`  
`pip install numpy`  
# How to run
`cd <GameFolderName> && python main.py`
